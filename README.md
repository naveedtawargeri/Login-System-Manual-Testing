\# Login System Manual Testing Project



\## Project Overview



This project demonstrates end-to-end Manual Testing of a Login Module.



\## Scope



The Login Module was tested to verify authentication functionality, field validations, UI behavior, and security-related scenarios.



\## Testing Types Covered



\* Positive Testing

\* Negative Testing

\* Validation Testing

\* Functional Testing

\* UI Testing

\* Security Testing



\## Test Artifacts



\* Test Plan

\* Test Scenarios

\* Test Cases

\* Test Execution Report

\* Bug Report

\* Test Coverage Report

\* Testing Type Coverage Report



\## Defects Identified



\* Username validation missing

\* Password masking issue

\* Forgot Password link broken

\* Username length validation issue

\* UI alignment issue

\* Account lockout issue
